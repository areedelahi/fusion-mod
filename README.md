# fusion
this neoforge mod for 1.21.4 allows two players to fuse into a single shared entity using a custom siamese system.
it manages shared inventory, movement, and interaction logic to make the fusion feel seamless and playable.
i implemented custom packet handling and mixins to synchronize inputs between the two players in real-time.
great for co-op challenges or just messing around with friends on a server.
built using java 21 and focuses on high-performance synchronization.
